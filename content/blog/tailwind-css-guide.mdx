# Why Use Tailwind CSS?

Tailwind CSS provides **utility classes** that allow you to style your application without writing custom CSS.

## Example:
```html
<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">
  Click me
</button>
